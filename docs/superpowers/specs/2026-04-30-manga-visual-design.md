# ASR Server — 日漫风视觉改版设计

## 设计决策

### 1. 背景系统
- 用户可自定义背景图（CSS 变量 `--bg-image`），固定附着
- 半透明暗色遮罩（`rgba(15,23,42,0.85)`）保证内容可读性
- 所有卡片/面板使用半透明背景 + backdrop-filter blur，透出背景

### 2. 色彩体系
| 角色 | 色值 | 用途 |
|------|------|------|
| 底色 | `#0f172a` | 页面底层 |
| 卡片色 | `rgba(30,41,59,0.85)` | 面板、表单 |
| 主色 | `#38bdf8` | 链接、高亮、活跃态 |
| 漫画粉 | `#ff6b9d` | 主要操作按钮 |
| 英雄蓝 | `#38bdf8 → #818cf8` | 次要操作按钮（渐变） |
| 错误红 | `#ff6b6b` | 危险/删除操作 |

### 3. 导航栏
- C 方案：毛玻璃悬浮
- `background: rgba(15,23,42,0.75)` + `backdrop-filter: blur(20px)`
- 底部极细线 `rgba(255,255,255,0.06)`

### 4. 卡片
- B 方案：实色微光
- `rgba(30,41,59,0.85)` 半透明底，hover 时增强光晕
- 8px 圆角，hover 时 `border-color` 变为主色半透明

### 5. 按钮 — 日漫漫画风
三种风格按语义使用：

| 风格 | 语义 | 样式 |
|------|------|------|
| 冲击波 (Impact) | 主要操作 | `#ff6b9d` 粉底 + `#0f172a` 粗轮廓 + 3px 阴影偏移 |
| 英雄 (Hero) | 次要操作 | 蓝紫渐变 + 白描边 + 阴影 |
| 线稿 (Outline) | 辅助/取消 | 透明底 + 描边 |

交互：按下时阴影缩小/消失，模拟"按入"物理感。

### 6. 字体
- 保留系统字体栈，`Noto Sans SC` 作为中文字体优先

### 7. 微交互
- 卡片 hover：边框变色 + 微光增加
- 按钮 press：阴影偏移减少（3px → 1px → 0px）
- 状态灯：绿色呼吸动画（已实现）
- 转写词逐字 fadeIn（已实现）
